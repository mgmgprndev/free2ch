Free2Ch Bot

Free2Ch Bot("bot") is Image Proxy System built by Free2Ch.net and hosted using Cloudflare Worker.
This allows to prevent to load directly malicious image links that logs IP, User-Agents, e.g.
The images proxied with this will not saved to server at all, and it is not possible to disallow specific URL or images to proxied for technical restriction.
This bot will only used purposes of protecting privacy of viewers of image.
For more information please Email to bot@free2ch.net, We will do our best.
